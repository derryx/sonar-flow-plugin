The should be no invocations of pub.flow:clearPipeline in non-toplevel services

The should be no invocations of pub.flow:clearPipeline because clearing the pipeline globally can have unwanted side-effects.

Noncompliant Code Example

  Invoke pub.flow:clearPipeline

Compliant Solution

  Remove invocation of pub.flow:clearPipeline and replace it with a MAP step to drop unwanted pipeline contents.
