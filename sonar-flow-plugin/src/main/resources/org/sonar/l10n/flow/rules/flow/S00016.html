Services with interface element 'REPEAT' need to be configured correctly. 

  Services with interface element 'REPEAT' step need to be configured correctly:
  	- The "Count" property needs to be configured. You should not set it to -1 (means NO LIMIT). This can lead to infinite loops.
  	- It is bad practice to leave the "Timeout" property empty

Noncompliant Code Example

Any non-compliant variant

Compliant Solution

	- Configure the "Count" property. You should not set it -1. This can lead to infinite loops!
  	- Configure the "Timeout" property.
