There are too many nested step levels.

Nesting too many step levels leads to hard to maintain and hard to debug code.

Noncompliant Code Example

   BRANCH
       SEQUENCE
           LOOP
               BRANCH
                   SEQUENCE
                        SEQUENCE

Compliant Solution

    Factor out parts of your code to other services to reduce the complexity of your flow service:
    
    BRANCH
        SEQUENCE
             INVOKE
